Real Number System

Structure and Fundamental Properties in Real Analysis

1. Structure of Number Systems

ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ

The real numbers ℝ include both rational and irrational numbers and form a continuous number system.

2. Completeness of ℝ

Theorem (Least Upper Bound Property).

Every nonempty subset S ⊂ ℝ that is bounded above has a least upper bound in ℝ.

Example

S = { x ∈ ℚ | x² < 2 }

This set has no supremum in ℚ, but in ℝ its supremum is √2. This illustrates the completeness of ℝ.

3. Archimedean Principle

Theorem. Let a, b ∈ ℝ with a > 0. Then there exists n ∈ ℕ such that

b < na

Proof

Case 1: b ≤ a

If b < a, take n = 1. Then b < a = na. If b = a, take n = 2. Then b < 2a = na.

Case 2: a < b

Suppose that no such natural number exists. Then for all n ∈ ℕ, we have na ≤ b.

Hence the set {na : n ∈ ℕ} is bounded above by b. By completeness of ℝ, let α = sup{na}.

Since a > 0, α - a < α. Thus α - a is not an upper bound, so there exists n ∈ ℕ such that

na > α - a

Adding a gives

(n+1)a > α

which contradicts that α is an upper bound. Therefore, there exists n ∈ ℕ such that b < na.

Example

Let a = 0.5 and b = 10. Then n = 21 gives na = 10.5 > 10.

4. ε-δ Definition of Limit

Definition.

∀ ε > 0, ∃ δ > 0 such that 0 < |x - a| < δ ⇒ |f(x) - L| < ε

This means that f(x) can be made arbitrarily close to L when x is sufficiently close to a.

Example

Let f(x) = x², a = 2. Then L = 4.

|x² - 4| = |x - 2||x + 2|

If |x - 2| < 1, then |x + 2| < 5. Hence,

|x² - 4| < 5|x - 2|

To make this less than ε, choose δ = ε/5.

5. Conclusion

The real number system is characterized by completeness and the Archimedean property. Using these, limits are rigorously defined by the ε-δ method, forming the foundation of real analysis.

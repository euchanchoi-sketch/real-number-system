Real Number System

해석학에서의 실수 체계를 간단히 정리한 페이지

1. 자연수 (N)

1, 2, 3, ... 과 같이 세는 데 사용하는 수의 집합

2. 정수 (Z)

..., -2, -1, 0, 1, 2, ... 와 같이 음수까지 포함한 집합

3. 유리수 (Q)

분수 형태로 나타낼 수 있는 수 (예: 1/2, -3/4)

4. 무리수

분수로 나타낼 수 없는 수 (예: √2, π)

5. 실수 (R)

유리수와 무리수를 모두 포함하는 수 체계

6. 완비성 (Completeness)

실수체계의 가장 중요한 성질로, 모든 유계 집합은 상한(최소 상계)을 가진다.
